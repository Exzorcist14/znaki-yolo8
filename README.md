# Знаки ПДД

Мобильное Android-приложение для распознавания дорожных знаков в реальном времени
с камеры телефона. Распознаёт **55 классов знаков ПДД**, отображает зелёную рамку
вокруг знака и **озвучивает его название голосом** на русском языке.

В основе — **YOLOv8 INT8 (TFLite)**, обученная на ~40 000 фотографий, и
**CameraX** для потокового анализа кадров.

## Как установить и пользоваться

Подробная пошаговая инструкция для конечного пользователя — в файле
[INSTALL.md](INSTALL.md). Краткая версия:

1. Скачайте APK по ссылке выше.
2. Откройте файл на Android-телефоне (минимум Android 7.0).
3. Если ОС предупредит об «установке из неизвестных источников» — разрешите для
   текущего приложения и подтвердите установку.
4. Запустите «Знаки ПДД», разрешите доступ к камере.
5. Наведите камеру на знак с расстояния ~1–3 метров. Появится зелёная рамка с
   подписью на русском, и приложение озвучит знак.

Шестерёнка справа сверху — настройки: озвучка, порог уверенности, тест голоса.

## Технологический стек

- **Kotlin**, минимальная цель Android API 24 (Android 7.0).
- **CameraX** — потоковый захват кадров с камеры.
- **TensorFlow Lite** — инференс модели YOLOv8 INT8 на устройстве, без интернета.
- **Custom NMS + временной стабилизатор** — фильтрация и сглаживание выхода YOLO,
  чтобы рамки не «мигали» от кадра к кадру.
- **Android TextToSpeech** — оффлайн озвучка распознанных знаков на русском языке
  (через Google TTS, с фолбэком на английский).

## Известные ограничения

Модель обучена на ограниченном наборе из 55 классов; визуально похожие
треугольные знаки могут путаться, а классы, которых нет в обучающем наборе
(например, «Подъём», «Боковой ветер», «Ограничение 40»), не распознаются
вообще — они не входят в эти 55. Для каждого реального применения требуется
дальнейшее расширение датасета и дообучение модели.
